CSS - Proprietà e Dimostrazioni

CSS (Cascading Style Sheets) è il linguaggio utilizzato per descrivere la presentazione di documenti HTML. Di seguito trovi una spiegazione delle principali proprietà di CSS con esempi pratici.

Colore del Testo

Puoi cambiare il colore del testo utilizzando la proprietà color:

p { color: blue; }

Margini e Padding

Margini e padding sono usati per creare spazi attorno agli elementi:

div { margin: 20px; padding: 10px; }

Background

Puoi aggiungere un colore di sfondo o un'immagine di sfondo con la proprietà background:

div { background-color: #f0f0f0; background-image: url('immagine.jpg'); }

Font e Testo

Le proprietà font-family, font-size e font-weight possono essere usate per stilizzare il testo:

p { font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; }

Bordi

Puoi aggiungere bordi agli elementi utilizzando la proprietà border:

div { border: 2px solid black; border-radius: 10px; }

Altezza e Larghezza

Le proprietà height e width sono utilizzate per definire le dimensioni di un elemento:

div { height: 100px; width: 200px; }

Display

La proprietà display è usata per definire il comportamento di un elemento (inline, block, flex, etc.):

div { display: flex; }

Posizionamento

La proprietà position consente di controllare il posizionamento degli elementi sulla pagina:

div { position: absolute; top: 50px; left: 100px; }

Ombre

Le ombre possono essere applicate agli elementi con la proprietà box-shadow per i box e text-shadow per il testo:

div { box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); } p { text-shadow: 2px 2px 5px grey; }

Flexbox

Flexbox è un potente metodo di layout CSS per disporre gli elementi:

.container { display: flex; justify-content: center; align-items: center; }

Grid

CSS Grid è un sistema di layout per creare griglie di elementi:

.grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; }

Transizioni e Animazioni

Le transizioni e le animazioni possono aggiungere effetti interattivi ai tuoi elementi:

div { transition: all 0.3s ease; } @keyframes esempio-animazione { from { opacity: 0; } to { opacity: 1; } }

Media Queries

Le media queries ti permettono di applicare stili diversi in base alle dimensioni dello schermo:

@media (max-width: 600px) { body { font-size: 14px; } }

Float

La proprietà float è utilizzata per far galleggiare gli elementi a destra o a sinistra:

img { float: left; margin-right: 10px; }

Visibilità

La proprietà visibility controlla se un elemento è visibile o nascosto, ma mantiene il suo spazio:

div { visibility: hidden; }
