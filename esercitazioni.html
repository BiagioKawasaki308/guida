Esercitazioni Front-End

In questa sezione troverai esercizi pratici per migliorare le tue abilità con HTML, CSS e JavaScript. Gli esercizi ti aiuteranno a rafforzare i concetti appresi nelle lezioni precedenti.

Esercizio 1: Creare una Pagina HTML Semplice

Creare una pagina HTML che includa:

  • Un titolo (<h1>).
  • Due paragrafi (<p>).
  • Un collegamento ipertestuale (<a>).

Esercizio 2: Stile con CSS

Aggiungere stili CSS alla pagina HTML creata nell'esercizio precedente. Includere:

  • Un colore di sfondo per la pagina.
  • Un colore per il testo dei paragrafi.
  • Stili per il titolo, come il colore e la dimensione del font.

Esercizio 3: Interazioni con JavaScript

Aggiungere funzionalità interattive alla pagina utilizzando JavaScript. Includere:

  • Un pulsante che, quando cliccato, mostra un messaggio di benvenuto.
  • Un evento che cambia il colore del titolo quando viene cliccato.

Esercizio 4: Layout con Flexbox

Creare un layout semplice utilizzando Flexbox. La pagina deve contenere:

  • Un header, un contenuto principale e un footer.
  • Il contenuto principale deve avere due colonne affiancate.

Esercizio 5: Utilizzo di Grid Layout

Creare un layout utilizzando CSS Grid che includa:

  • Un header, una barra laterale, un contenuto principale e un footer.
  • Disporre gli elementi in una griglia con due colonne e due righe.

Esercizio 6: Validazione di un Form con JavaScript

Creare un modulo HTML con campi per nome, email e un pulsante di invio. Utilizzare JavaScript per:

  • Verificare che il nome e l'email non siano vuoti.
  • Mostrare un messaggio di errore se i campi non sono correttamente compilati.

Esercizio 7: Animazioni con CSS

Creare una semplice animazione con CSS che faccia muovere un elemento da sinistra a destra sulla pagina.

  • Utilizzare @keyframes per definire l'animazione.
  • Applicare l'animazione a un elemento <div> con classe "box".
