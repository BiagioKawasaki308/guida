JavaScript - Fondamenti del Linguaggio

JavaScript è un linguaggio di programmazione utilizzato per aggiungere funzionalità interattive alle pagine web. Di seguito troverai una guida sui principali concetti di JavaScript con esempi pratici per facilitare l'apprendimento.

Variabili

Le variabili vengono utilizzate per memorizzare dati. Puoi dichiarare una variabile in JavaScript usando var, let o const:

let nome = "Mario";
const eta = 30;
var città = "Roma";

Tipi di Dati

JavaScript supporta diversi tipi di dati, tra cui stringhe, numeri, booleani, oggetti e array:

let stringa = "Ciao!";
let numero = 42;
let booleano = true;
let oggetto = { nome: "Mario", età: 30 };
let array = [1, 2, 3, 4];

Funzioni

Le funzioni vengono utilizzate per eseguire delle operazioni. Puoi dichiarare una funzione utilizzando la parola chiave function:

function saluta() {
  console.log("Ciao, mondo!");
}

saluta();

Operatori

Gli operatori vengono utilizzati per eseguire operazioni su variabili e valori. JavaScript supporta operatori aritmetici, di assegnazione, di confronto e logici:

let somma = 5 + 3;
let uguale = (5 == "5");
let maggiore = (10 > 5);
let logico = (true && false);

Condizioni

Le istruzioni condizionali permettono di eseguire codice diverso in base a una certa condizione, utilizzando if, else if, e else:

let ora = 20;
if (ora < 12) {
  console.log("Buongiorno!");
} else if (ora < 18) {
  console.log("Buon pomeriggio!");
} else {
  console.log("Buonasera!");
}

Cicli

I cicli vengono utilizzati per ripetere un blocco di codice più volte. JavaScript supporta for, while, e do...while:

for (let i = 0; i < 5; i++) {
  console.log("Iterazione: " + i);
}

Eventi

JavaScript può rispondere a eventi degli utenti, come click su un pulsante:

<button onclick="mostraMessaggio()">Cliccami!</button>

<script>
function mostraMessaggio() {
  alert("Hai cliccato il pulsante!");
}
</script>

Oggetti e Metodi

Gli oggetti in JavaScript sono collezioni di proprietà e metodi. Puoi definire un oggetto e accedere alle sue proprietà e metodi:

let persona = {
  nome: "Luca",
  età: 25,
  saluta: function() {
    console.log("Ciao, mi chiamo " + this.nome);
  }
};

persona.saluta();

Array e Metodi degli Array

Gli array sono liste di elementi. JavaScript offre diversi metodi per manipolare gli array, come push, pop, shift, unshift, e forEach:

let numeri = [1, 2, 3];
numeri.push(4);
numeri.forEach(function(numero) {
  console.log(numero);
});

Async e Await

JavaScript supporta la programmazione asincrona usando async e await, che permettono di gestire operazioni asincrone in modo più leggibile:

async function ottieniDati() {
  let risposta = await fetch('https://api.example.com/dati');
  let dati = await risposta.json();
  console.log(dati);
}

ottieniDati();

DOM (Document Object Model)

JavaScript può essere utilizzato per manipolare il DOM, cioè la struttura di una pagina HTML, per creare interazioni dinamiche:

document.getElementById("titolo").innerHTML = "Nuovo Titolo";
document.querySelector(".classe").style.color = "blue";
