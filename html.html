HTML - Fondamenta del Web

HTML (HyperText Markup Language) è il linguaggio usato per creare la struttura delle pagine web. Di seguito trovi una spiegazione delle principali regole di HTML con esempi pratici.

Introduzione

HTML è un linguaggio di markup utilizzato per definire la struttura e il contenuto di una pagina web. Ogni elemento HTML è rappresentato da un tag, che può avere attributi specifici per configurare il comportamento o l'aspetto dell'elemento.

Editor HTML

Per scrivere codice HTML puoi utilizzare vari editor, da semplici editor di testo come il Blocco Note fino ad ambienti di sviluppo più complessi come VS Code, Sublime Text o Atom.

Attributi HTML

Gli attributi HTML sono utilizzati per fornire ulteriori informazioni sugli elementi. Ad esempio, il tag <a> ha un attributo href per specificare l'URL:

<a href="https://www.example.com" target="_blank">Vai a Example</a>

Paragrafi e Titoli

I paragrafi sono definiti con il tag <p>, mentre i titoli vanno da <h1> a <h6> per specificare l'importanza del titolo, dove <h1> è il più importante.

<h1>Titolo Principale</h1>
<p>Questo è un paragrafo.</p>

Formattazione Testo

Puoi utilizzare vari tag per formattare il testo, come <b> per il grassetto, <i> per il corsivo e <u> per sottolineare il testo.

<p>Questo è un <b>testo in grassetto</b>, <i>corsivo</i>, e <u>sottolineato</u>.</p>

Quotazioni e Commenti

Per aggiungere citazioni puoi usare il tag <blockquote>. I commenti HTML, che non vengono visualizzati dall'utente, possono essere aggiunti con <!-- commento -->.

<blockquote>Questa è una citazione.</blockquote>
<!-- Questo è un commento HTML -->

Collegamenti e Immagini

I collegamenti e le immagini sono elementi essenziali del web. Il tag <img> è utilizzato per incorporare immagini, mentre il tag <a> crea link ipertestuali.

<a href="https://www.example.com">Link a Example</a>
<img src="immagini/logo.png" alt="Logo del sito">

Favicon

Una favicon è una piccola icona che rappresenta il sito web nel browser. Puoi aggiungerla con il seguente tag nel <head>:

<link rel="icon" href="favicon.ico" type="image/x-icon">

Div e Span

Il tag <div> è usato per raggruppare blocchi di contenuto, mentre il tag <span> è usato per stilizzare specifiche parti di testo inline.

<div>Questa è una sezione di contenuto.</div>
<span style="color: red;">Questo è testo colorato</span>

JavaScript nel HTML

Per aggiungere funzionalità interattive alla tua pagina HTML puoi utilizzare JavaScript. Puoi inserire il codice JavaScript direttamente nella pagina con il tag <script>.

<script>
  alert('Benvenuto nella mia pagina!');
</script>

File Paths

Il percorso di un file (file path) può essere assoluto o relativo. Ad esempio, il percorso ./immagini/logo.png fa riferimento a una directory locale.

Responsive Design

Per rendere il tuo sito web adattabile a diversi dispositivi, puoi utilizzare tecniche di responsive design come le media query e attributi di larghezza in percentuale.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<style>
  @media (max-width: 600px) {
    body { font-size: 14px; }
  }
</style>

Tabelle

Le tabelle sono utili per visualizzare dati strutturati in formato tabellare. Ecco un esempio:

<table>
  <tr>
    <th>Nome</th>
    <th>Età</th>
  </tr>
  <tr>
    <td>Mario</td>
    <td>30</td>
  </tr>
</table>

Liste

HTML supporta sia liste ordinate che non ordinate per organizzare il contenuto:

<h3>Lista Ordinata</h3>
<ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ol>

<h3>Lista Non Ordinata</h3>
<ul>
  <li>Elemento A</li>
  <li>Elemento B</li>
</ul>

Form

I form sono usati per raccogliere dati dagli utenti:

<form action="/invia-dati" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">

  <label for="email">Email:</label>
  <input type="email" id="email" name="email">

  <input type="submit" value="Invia">
</form>

Metatag

I metatag sono usati per fornire metadati sul documento HTML. Ad esempio, il tag <meta> può specificare la descrizione, il set di caratteri, e altre informazioni:

<meta charset="UTF-8">
<meta name="description" content="Questa è una pagina di esempio">

Attributi Globali

Gli attributi globali possono essere usati su qualsiasi elemento HTML. Ad esempio, id, class, e style sono attributi globali che possono essere applicati per identificare elementi o aggiungere stili:

<div id="sezione1" class="contenitore" style="background-color: lightblue;">Contenuto della sezione</div>

IFrame

Il tag <iframe> viene utilizzato per incorporare un'altra pagina web all'interno di una pagina HTML. Ad esempio, puoi incorporare un video di YouTube o un'altra pagina web:

<iframe src="https://www.youtube.com/embed/dQw4w9WgXcQ" width="560" height="315"></iframe>

Attributo Data-*

L'attributo data-* viene utilizzato per memorizzare dati personalizzati sugli elementi HTML, spesso usato in combinazione con JavaScript per interazioni dinamiche:

<button id="myButton" data-info="Esempio di Data">Cliccami</button>

Sezione Head e Headings

La sezione <head> è la parte del documento HTML dove vengono inseriti titoli, metatag e collegamenti ai file CSS o JavaScript. Gli headings, come <h1> - <h6>, definiscono i titoli principali e secondari:

<head>
  <title>Titolo della Pagina</title>
  <link rel="stylesheet" href="styles.css">
</head>
